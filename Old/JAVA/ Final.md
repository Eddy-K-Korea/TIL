# Final

### 종류
1. 변수에 사용
2. 메소드에 사용
3. 클래스에 사용

> ### 1. 변수에 사용

```java
final int MEMBER_NUMBER = 100;
```
변수에 final을 붙여 줌으로써 변수가 상수로 적용이된다.

변하지 않는 값을 변수에 대입하고자 할때 사용된다.

> ### 2. 메소드에 사용

```java
public final void method() {	
}
```
final을 통해 메소드의 Over Ride를 제한 한다.

> ### 3. 클래스에 사용

```java
final class superClass{

}
```
상속을 제한하는 클래스에 final 사용이다.
